- **Backend**: Express, FastAPI, Gin などのバックエンドフレームワーク
- **CLI / Tools**: CLIツール、開発者向けツール
- **Infrastructure**: Docker, Kubernetes, Terraform など
- **Summary**: 比較表、まとめ
